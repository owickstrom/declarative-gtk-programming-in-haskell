# Declarative GTK+ Programming in Haskell

A talk given at LambdAle in London, September 1st, 2018.

## Building

This requires:

* Make
* TexLive
* Pandoc 2.x
* pandoc-include-code
* pandoc-emphasize-code >= 0.2.4

Run:

```
make
```
